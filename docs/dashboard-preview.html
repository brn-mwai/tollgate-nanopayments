How CCTP off-ramp works
Circle Bridge Kit · native burn + mint
1. Burn · USDC burned on Arc via Circle Wallets.
2. Attestation · Circle signs the burn receipt off-chain.
3. Mint · USDC minted on destination chain (Base, Ethereum, Solana).
4. Off-ramp · route to your preferred fiat on-ramp (optional).

Install the SDK
One line of middleware. Start charging AI bots per request.
Hono
Node 20+, Bun, or Deno
// npm i @tollgate/hono hono
import { Hono } from 'hono'
import { tollgate } from '@tollgate/hono'

const app = new Hono()

app.use('/article/*', tollgate({
  siteId:   process.env.TOLLGATE_SITE_ID,
  price:    '500uUSDC',        // $0.0005 per request
  chain:    'arc-testnet',
  receipt:  { ttl: 300 }        // 5-minute HMAC cache
}))

app.get('/article/:id', c => c.json({ body: '...' }))

export default app

Cloudflare Worker
wrangler 3.x
// npm i @tollgate/cloudflare-worker
import { tollgate } from '@tollgate/cloudflare-worker'

export default {
  async fetch(request, env, ctx) {
    const gated = await tollgate({
      siteId:   env.TOLLGATE_SITE_ID,
      price:    '500uUSDC',
      chain:    'arc-testnet',
      receipt:  { ttl: 300 },
      kv:       env.TOLLGATE_KV   // for nonce dedup
    })(request)

    if (gated) return gated
    return new Response('article body')
  }
}
